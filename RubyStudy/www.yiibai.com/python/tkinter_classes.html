4. Tkinter类 Classes

 小部件类

Tkinter支持15个核心小部件:

Table 4-1. Tkinter Widget Classes

Widget 描述
Button 一个简单的按钮，用于执行命令或进行其他操作。
Canvas 结构化的图形。这个小工具可以用来绘制图表和图形，创建图形编辑器，并实现自定义的部件。
Checkbutton 代表一个变量，可以有两个不同的值。按一下按钮值之间的切换。
Entry 一个文本输入字段。
Frame 一个容器组件。框架可以有一个边框和背景，并用于进行分组的其他窗口小部件，当创建一个应用程序或对话框布局。
Label 显示文本或图像。
Listbox 以列表形式显示的替代品。可配置的列表框，单选按钮或核对表的行为。
Menu 一个菜单窗格中。用于实现下拉菜单和弹出菜单。
Menubutton 一个菜单按钮。用于实现下拉菜单。
Message 显示文本。相似的标签小部件，但可以自动换行的文本以一个给定的宽度或宽高比。
Radiobutton 表示一个变量，它可以有许多值之一的一个值。按一下按钮，设置的变量值，并清除所有其他单选按钮具有相同的变量相关联的。
Scale 允许你设置一个数值，通过拖动“滑块”。
Scrollbar 画布，输入，列表框和文本部件使用标准的滚动条。
Text 格式化文本显示。您可以显示和编辑文本的各种样式和属性。同时还支持嵌入图像和窗口。
Toplevel 作为一个独立的，顶层的窗口显示的容器部件。


还要注意的是有没有小部件类层次结构的Tkinter，所有的widget类是在继承树中的兄弟姐妹。
所有这些部件提供的杂项和几何的管理方法，配置管理方法，以及其他方法所定义的小部件本身。此外，根类还提供了窗口管理器的界面。这意味着，一个典型的小部件类提供了约150种方法。


Mixins

Tkinter模块提供的类对应于不同类型的组件在Tk，混入其他一些辅助类（一个mixin是一类与其他类使用多重继承相结合）。当你使用Tkinter不应该直接访问的mixin类。

 Geometry mixins

混入Grid, Pack并把类作为窗口小部件类。他们提供的各种几何图形管理器的访问，也可以通过委托。

Table 4-2. Geometry Mixins

Manager Description
Grid The grid geometry manager allows you to create table-like layouts, by organizing the widgets in a 2-dimensional grid. To use this geometry manager, use the grid method.
Pack The pack geometry manager lets you create a layout by "packing" the widgets into a parent widget, by treating them as rectangular blocks placed in a frame. To use this geometry manager for a widget, use the pack method on that widget to set things up.
Place The place geometry manager lets you explicitly place a widget in a given position. To use this geometry manager, use the placemethod.
