Hello, Tkinter例子

Hello, Tkinter例子，正如你所知道的，每一个严肃的的教程应该从一个“hello world”型的例子。在这个概述中，我们将告诉你不只是一个例子，但有两个。

2. Hello, Tkinter程序例子

长话少说吧，接下的时间，我们看一些代码。
正如你所知道的，每一个严肃的的教程应该从一个“hello world”型的例子。在这个概述中，我们将告诉你不只是一个例子，但有两个。
首先，让我们来看看在一个漂亮的最低版本：

例子2-1. 我们的第一个Tkinter程序 

# File: hello1.py

from Tkinter import *

root = Tk()

w = Label(root, text="Hello, world!")
w.pack()

root.mainloop()

执行以上的程序例子

要运行该程序，运行的脚本像往常一样（注意，可能有的朋友在window机器上运行，那么可以考虑用Ulipad开发工具）： 

$ python hello1.py

详细说明

首先，我们导入Tkinter模块。它包含了所有的类，函数和其他需要的东西到了Tk工具包。在大多数情况下，你可以简单地导入一切从Tkinter模块的命名空间： 

    from Tkinter import *

要初始化Tkinter，我们要创建一个Tk root部件。这是一个普通的窗口，提供由你的窗口管理器的标题栏和其他装饰。你应该只为每个程序创建一个root部件，它必须建立在任何其他的部件。 

    root = Tk()

接下来，我们创建了一个标签控件作为一个子到根窗口： 

    w = Label(root, text="Hello, world!")
    w.pack()

一个标签控件可以显示文本或图标或其他图像。在这种情况下，我们使用的文字选项来指定要显示的文本。接下来，，我们称之为packmethod这个小工具，告诉它本身大小，以适应给定的文本，使自己可见。但在此之前，我们必须进入Tkinter的事件循环： 

    root.mainloop()

该计划将留在事件循环，直到我们关闭窗口。事件循环不只是处理事件（如鼠标点击和按键），或者从用户的窗口系统（如重绘事件和窗口配置消息），它也处理排队的Tkinter本身的操作。在这些操作中的的几何管理（排队的包法）和显示更新。这也意味着，应用程序窗口将不会出现在你面前进入主循环。
