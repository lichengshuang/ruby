1. 什么是Tkinter?

Tkinter模块（“TK接口”）是标准的Python接口Tk的GUI工具包Scriptics（以前的Sun实验室开发）。

Tk和Tkinter可在大多数Unix平台上，以及在Windows和Macintosh系统。从8.0版本开始，传统知识在所有平台上提供了本机的外观和感觉。

Tkinter由大量的模块。Tk接口位于一个二进制模块名为_tkinter（这是Tkinter的早期版本中）。该模块包含低级别的接口Tk，不应该直接使用应用程序的程序员。它通常是一个共享库（或DLL），但可能在某些情况下，用Python解释器静态链接。

在除了Tk接口模块，Tkinter的Python模块包括一个数。两个最重要的模块是的Tkinter模块本身，和一个模块调用Tkconstants。前者会自动导入后者，因此使用Tkinter，所有你需要做的是导入一个模块： 

    import Tkinter

或者，更多的时候使用以下方式： 

    from Tkinter import *
