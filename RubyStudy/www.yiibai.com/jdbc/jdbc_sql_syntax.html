JDBC SQL语法

JDBC SQL语法实例代码教程 - 结构化查询语言（SQL）是一种标准化的语言，可让您执行操作的数据库，如创建条目，阅读内容，更新内容，并删除项目。

结构化查询语言（SQL）是一种标准化的语言，可让您执行操作的数据库，如创建条目，阅读内容，更新内容，并删除项目。

SQL支持任何数据库，它可以让你编写数据库代码独立的底层数据库。

本教程简要介绍了SQL，这是一个先决条件，了解JDBC的概念。本教程为您提供了足够的SQL的创建，读取，更新和删除数据库中的数据（通常简称为CRUD操作）。

想详细的了解SQL，可以看看MySQL教程

创建数据库:

CREATE DATABASE语句用于创建一个新的数据库。语法是： 

SQL> CREATE DATABASE DATABASE_NAME;

例子:

下面的SQL语句创建了一个名为EMP的数据库： 

SQL> CREATE DATABASE EMP;

删除数据库:

DROP DATABASE语句用于删除一个现有的数据库。语法是： 

SQL> DROP DATABASE DATABASE_NAME;

注: 要创建或删除数据库，在数据库服务器上，应该有管理员权限。要小心，删除的数据库将会所有存储在数据库中数据删除。

创建表:

CREATE TABLE语句用于创建一个新表。语法是： 

SQL> CREATE TABLE table_name
(
   column_name column_data_type,
   column_name column_data_type,
   column_name column_data_type
   ...
);

例子:

下面的SQL语句创建了一个名为Employees的表有四个列： 

SQL> CREATE TABLE Employees
(
   id INT NOT NULL,
   age INT NOT NULL,
   first VARCHAR(255),
   last VARCHAR(255),
   PRIMARY KEY ( id )
);

删除表:

DROP TABLE语句用于删除一个现有的表。语法是： 

SQL> DROP TABLE table_name;

例子:

下面的SQL语句删除一个名为Employees表： 

SQL> DROP TABLE Employees;

插入数据:

为INSERT语法看起来类似于以下内容，其中column1，column1，等代表新的数据出现在各列： 

SQL> INSERT INTO table_name VALUES (column1, column2, ...);

例子:

下面的SQL INSERT语句插入新行的员工到先前创建的数据库表： 

SQL> INSERT INTO Employees VALUES (100, 18, 'Zara', 'Ali');

SELECT Data:

使用SELECT语句从数据库中检索数据。 SELECT的语法是： 

SQL> SELECT column_name, column_name, ...
     FROM table_name
     WHERE conditions;

在WHERE子句中可以使用比较操作符例如=，=，<，>，<=，>=，以及BETWEEN和LIKE操作符。

例子:

下面的SQL语句选择Employees表中的ID列是100岁，第一次和最后一列： 

SQL> SELECT first, last, age 
     FROM Employees 
     WHERE id = 100;

下面的SQL语句选择的年龄，第一次和最后一列从Employees表中第一列包含www.yiibai.com： 

SQL> SELECT first, last, age 
     FROM Employees 
     WHERE first LIKE '%www.yiibai.com%';

更新数据:

使用UPDATE语句来更新数据。 UPDATE的语法是： 

SQL> UPDATE table_name
     SET column_name = value, column_name = value, ...
     WHERE conditions;

在WHERE子句中可以使用比较操作符例如=，=，<，>，<=，>=，以及BETWEEN和LIKE操作符。

例子:

下面的SQL UPDATE语句更改ID为100的雇员的年龄列： 

SQL> UPDATE Employees SET age=20 WHERE id=100;

删除数据:

使用DELETE语句删除表中的数据。 DELETE的语法是： 

SQL> DELETE FROM table_name WHERE conditions;

在WHERE子句中可以使用比较操作符例如=，=，<，>，<=，>=，以及BETWEEN和LIKE操作符。

例子:

下面的SQL DELETE语句删除ID为100的雇员的记录： 

SQL> DELETE FROM Employees WHERE id=100;
