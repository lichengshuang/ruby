Perl语法概述

Perl语法概述，Perl语法实例，Perl语法例子，Perl代码实例 -一个Perl脚本或程序，包括一个或多个语句。

一个Perl的脚本或程序由一个或多个语句，这些语句只是写在一个简单脚本，没有必要有一个main()函数或类似那种东西。
Perl语句以分号结束: 

print "Hello, world";

注释开始与一个hash符号和运行到该行的末尾: 

# This is a comment

空白是无关紧要: 

print      "Hello, world";

...除了内部引用的字符串: 

# this would print with a linebreak in the middle
print "Hello
          world";

围绕文字字符串可用于双引号或单引号: 

print "Hello, world";
print 'Hello, world';

然而，只有“插补”变量和特殊字符，如换行符（\ n）双引号: 

print "Hello, $name\n";     # works fine

print 'Hello, $name\n';     # prints $name\n literally

数字不需要引号: 

print 42;

您可以使用括号函数的参数或省略他们（根据您个人喜好），他们只偶尔需要区分优先性问题，以下两个语句产生同样的结果。 

print("Hello, world\n");
print "Hello, world\n";

Perl的文件扩展名:

Perl脚本可以使用任何简单文本编辑器。有几个方案可供不同类型的平台，有许多在网络上可供下载的程序设计。

无论您选择使用的程序（编辑器），一个Perl文件必须以.pl（perl等）文件名保存，以文件的扩展名被承认作为一个正常运作的Perl脚本。文件名可以包含数字，符号和字母，但不能包含空格，使用下划线（_）代替空格也是可以的。

第一个PERL程序:

假设你已经在Unix提示符，现在hello.pl使用vi或vim编辑器打开一个文本文件和内部文件的以下行。 

#!/usr/bin/perl

# This will print "Hello, World - by www.yiibai.com" on the screen
print "Hello, world ";

#!/usr/bin 是您已经安装了PERL的路径

执行Perl脚本:

在你执行你的脚本时，是可能需要改变脚本文件的模式和执行权限，一般设定的0755。

现在从Unix命令行运行hello.pl的Perl程序，发出以下命令在您的UNIX提示符$: 

$perl hello.pl

这将产生以下结果: 

Hello, World

Perl命令行标志:

命令行标志影响Perl如何运行程序. 

$perl -v

这将产生以下结果: 

This is perl, v5.001 built for i386-linux-thread-multi
................

你可以在命令行使用-e选项，它可以让你的命令行执行Perl语句。 

$perl -e 'print 4;\n'
RESULT: 4
$perl -e "print 'Hello World!\n - www.yiibai.com";'
RESULT: Hello World!
