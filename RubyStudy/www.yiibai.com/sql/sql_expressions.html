SQL表达式

SQL表达式实例子代码教程 - SQL表达式可以是一个组合的一个或多个值，运算符和SQL函数的值计算。

表达式可以是一个组合的一个或多个值，运算符和SQL函数的值计算。

SQL表达式与公式类似，都写在查询语言。也可以用来查询数据库，为特定的数据集。

语法:

考虑的SELECT语句的基本语法如下： 

SELECT column1, column2, columnN 
FROM table_name 
WHERE [CONTION|EXPRESSION];

有许多不同类型的SQL的表达，这下面会提到：

SQL - 布尔表达式:

SQL布尔表达式匹配的单值的基础上获取数据。以下是语法： 

SELECT column1, column2, columnN 
FROM table_name 
WHERE SINGLE VALUE MATCHTING EXPRESSION;

考虑CUSTOMERS表中有以下记录： 

SQL> SELECT * FROM CUSTOMERS;
+----+----------+-----+-----------+----------+
| ID | NAME     | AGE | ADDRESS   | SALARY   |
+----+----------+-----+-----------+----------+
|  1 | Ramesh   |  32 | Ahmedabad |  2000.00 |
|  2 | Khilan   |  25 | Delhi     |  1500.00 |
|  3 | kaushik  |  23 | Kota      |  2000.00 |
|  4 | Chaitali |  25 | Mumbai    |  6500.00 |
|  5 | Hardik   |  27 | Bhopal    |  8500.00 |
|  6 | Komal    |  22 | MP        |  4500.00 |
|  7 | Muffy    |  24 | Indore    | 10000.00 |
+----+----------+-----+-----------+----------+
7 rows in set (0.00 sec)

下面是简单的例子来说明SQL布尔表达式的用法： 

SQL> SELECT * FROM CUSTOMERS WHERE SALARY = 10000;
+----+-------+-----+---------+----------+
| ID | NAME  | AGE | ADDRESS | SALARY   |
+----+-------+-----+---------+----------+
|  7 | Muffy |  24 | Indore  | 10000.00 |
+----+-------+-----+---------+----------+
1 row in set (0.00 sec)

SQL - 数值表达式:

该表达式用于在任何查询执行任何数学运算。以下是语法： 

SELECT numerical_expression as  OPERATION_NAME
[FROM table_name
WHERE CONDITION] ;

这里的数值表达式，用于数学表达式或任何公式。以下是一个简单的例子显示使用SQL数值表达式： 

SQL> SELECT (15 + 6) AS ADDITION
+----------+
| ADDITION |
+----------+
|       21 |
+----------+
1 row in set (0.00 sec)

有几个内置的功能，如SUM()，AVG()，COUNT()是已知的对一个表或一个特定的表列的汇总数据计算执行。 

SQL> SELECT COUNT(*) AS "RECORDS" FROM CUSTOMERS; 
+---------+
| RECORDS |
+---------+
|       7 |
+---------+
1 row in set (0.00 sec)

SQL - 日期表达式:

日期表达式返回当前系统日期和时间值： 

SQL>  SELECT CURRENT_TIMESTAMP;
+---------------------+
| Current_Timestamp   |
+---------------------+
| 2009-11-12 06:40:23 |
+---------------------+
1 row in set (0.00 sec)

另一个日期的表达式如下： 

SQL>  SELECT  GETDATE();;
+-------------------------+
| GETDATE                 |
+-------------------------+
| 2009-10-22 12:07:18.140 |
+-------------------------+
1 row in set (0.00 sec)
