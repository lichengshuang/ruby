SQL GROUP BY子句

SQL GROUP BY子句实例代码教程 - GROUP BY子句中使用SQL SELECT语句相同的数据分组排列与协作。

GROUP BY子句中使用SQL SELECT语句相同的数据分组排列与协作。

GROUP BY子句如下的SELECT语句中的WHERE子句和ORDER BY子句之前。

语法:

GROUP BY子句的基本语法如下所示。 GROUP BY子句中必须遵循的条件，必须先在WHERE子句中的ORDER BY子句如果使用。 

SELECT column1, column2
FROM table_name
WHERE [ conditions ]
GROUP BY column1, column2
ORDER BY column1, column2

例子:

考虑CUSTOMERS表中有以下记录： 

+----+----------+-----+-----------+----------+
| ID | NAME     | AGE | ADDRESS   | SALARY   |
+----+----------+-----+-----------+----------+
|  1 | Ramesh   |  32 | Ahmedabad |  2000.00 |
|  2 | Khilan   |  25 | Delhi     |  1500.00 |
|  3 | kaushik  |  23 | Kota      |  2000.00 |
|  4 | Chaitali |  25 | Mumbai    |  6500.00 |
|  5 | Hardik   |  27 | Bhopal    |  8500.00 |
|  6 | Komal    |  22 | MP        |  4500.00 |
|  7 | Muffy    |  24 | Indore    | 10000.00 |
+----+----------+-----+-----------+----------+

如果你想知道每个客户的薪金总额，然后GROUP BY查询将如下所示： 

SQL> SELECT NAME, SUM(SALARY) FROM CUSTOMERS
     GROUP BY NAME;

这将产生以下结果： 

+----------+-------------+
| NAME     | SUM(SALARY) |
+----------+-------------+
| Chaitali |     6500.00 |
| Hardik   |     8500.00 |
| kaushik  |     2000.00 |
| Khilan   |     1500.00 |
| Komal    |     4500.00 |
| Muffy    |    10000.00 |
| Ramesh   |     2000.00 |
+----------+-------------+
