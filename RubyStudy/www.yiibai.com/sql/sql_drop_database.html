SQL DROP或删除数据库

SQL DROP或删除数据库实例代码教程 - SQL语句 DROP DATABASE用于删除任何现有的数据库中的SQL模式。

SQL语句 DROP DATABASE用于删除任何现有的数据库中的SQL模式。

语法:

DROP DATABASE语句的基本语法如下： 

DROP DATABASE DatabaseName;

在RDBMS，总是数据库名称应该是唯一的。

例子:

如果你要删除一个现有的数据库<testDB>的，然后DROP DATABASE语句如下： 

SQL> DROP DATABASE testDB;

注: 使用此操作之前，要小心，因为删除一个现有的数据库，将导致失去存储在数据库中完整的信息。

请确保您具有管理员权限之前将任何数据库，一旦一个数据库被删除，可以检查它的数据库列表如下： 

SQL> SHOW DATABASES;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| AMROOD             |
| TUTORIALSPOINT     |
| mysql              |
| orig               |
| test               |
+--------------------+
6 rows in set (0.00 sec)
