MySQL更新查询

MySQL UPDATE更新查询实例代码教程-有可能要对MySQL表中的现有数据需要进行修改。可以通过使用SQL UPDATE命令。这将修改任何MySQL表中的任何字段的值。

有可能要对MySQL表中的现有数据需要进行修改。可以通过使用SQL UPDATE命令。这将修改任何MySQL表中的任何字段的值。

语法:

下面是通用的SQL语法的UPDATE命令来修改数据到MySQL表： 

UPDATE table_name SET field1=new-value1, field2=new-value2
[WHERE Clause]

  • 您可以更新一个或多个字段。

  • 您可以使用WHERE子句指定条件。

  • 您可以一次性将一个表中的值更新。

WHERE子句是非常有用的，当想更新选定的表中的行。

从命令提示符更新数据：

这将使用WHERE子句更新选定的数据转换成MySQL表tutorials_tbl的SQL UPDATE命令

实例:

下面的例子将更新tutorial_title字段中一个tutorial_id为3的记录。 

root@host# mysql -u root -p password;
Enter password:*******
mysql> use TUTORIALS;
Database changed
mysql> UPDATE tutorials_tbl 
    -> SET tutorial_title='Learning JAVA' 
    -> WHERE tutorial_id=3;
Query OK, 1 row affected (0.04 sec)
Rows matched: 1  Changed: 1  Warnings: 0

mysql>

使用PHP脚本更新数据：

可以使用SQL UPDATE命令，带有或不带WHERE子句PHP函数请求mysql_query()。此功能将执行类似的方式是在mysql>提示符下执行的SQL命令。

例子:

试试下面的例子为一个tutorial_id为3的记录，以更新tutorial_title的字段。 

<?php
$dbhost = 'localhost:3036';
$dbuser = 'root';
$dbpass = 'rootpassword';
$conn = mysql_connect($dbhost, $dbuser, $dbpass);
if(! $conn )
{
  die('Could not connect: ' . mysql_error());
}
$sql = 'UPDATE tutorials_tbl
        SET tutorial_title="Learning JAVA"
        WHERE tutorial_id=3';

mysql_select_db('TUTORIALS');
$retval = mysql_query( $sql, $conn );
if(! $retval )
{
  die('Could not update data: ' . mysql_error());
}
echo "Updated data successfully\n";
mysql_close($conn);
?>
