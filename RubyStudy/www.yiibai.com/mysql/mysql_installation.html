MySQL安装(Window&Linux)

MySQL安装(Window&Linux)实例教程-

下载MySQL:

在MySQL下载。选择版本号forMySQL社区服务器，尽可能准确地想要的版本。

在Linux/Unix上安装MySQL 

在Linux系统上安装MySQL推荐的方法是通过RPM。 MySQL AB公司在其网站上下载以下RPM：

  • MySQL - MySQL数据库服务器，管理数据库和表，控制用户访问和处理SQL查询。

  • MySQL-client - MySQL客户端程序，这使得它可以连接到交互服务器。

  • MySQL-devel -库和头文件，编译其他程序使用MySQL时派上用场。

  • MySQL-shared - 共享库的MySQL客户端

  • MySQL-bench - MySQL数据库服务器的标准和性能测试工具。

在这里列出的MySQL的RPM都是建立在SuSE Linux系统，但他们通常会与其他Linux版本没有任何问题。

现在，请按照以下步骤进行安装：

  1. 用root用户登录到系统。

  2. 切换到包含RPM的目录：

  3. 通过执行下面的命令来安装MySQL数据库服务器。请记住RPM文件名替换文件名中用斜体表示。 

    [root@host]# rpm -i MySQL-5.0.9-0.i386.rpm

    上面的命令负责安装MySQL服务器，创建一个MySQL的用户，创造必要的配置，并自动启动MySQL服务器。

    可以找到所有的MySQL相关的二进制文件在/usr/bin和/usr/sbin目录。所有的表和数据库将被创建在/var/lib/mysql目录。

  4. 这是可选的，但建议以安装其余的RPM包步骤相同： 

    [root@host]# rpm -i MySQL-client-5.0.9-0.i386.rpm
[root@host]# rpm -i MySQL-devel-5.0.9-0.i386.rpm
[root@host]# rpm -i MySQL-shared-5.0.9-0.i386.rpm
[root@host]# rpm -i MySQL-bench-5.0.9-0.i386.rpm

Windows上安装MySQL:

在任何版本的Windows的默认安装是现在比以前更容易，因为MySQL就整齐地打包安装程序。只需下载安装包，解压缩它的任何地方，并运行setup.exe。

默认安装程序setup.exe将引导通过简单的过程，默认情况下会安装在C\ mysql目录。

测试服务器击发的方式从命令提示符第一次。 mysqld服务器的位置，这可能是C:\mysql\bin，然后键入： 

mysqld.exe --console

注: 如果是在NT上，那么使用mysqld-nt.exe而不是mysqld.exe

如果一切顺利的话，会看到一些有关启动和InnoDB的消息。如果没有，可能有一个权限问题。请确保该目录保存数据访问，任何用户可能是mysql的数据库进程下运行。

MySQL不会将自身添加到“开始”菜单，没有特别漂亮的图形用户界面的方法来停止服务器。因此，如果倾向于通过双击的mysqld可执行文件启动服务器，应该记得要使用mysqladmin停止用手动的方式，可以使用任务管理器或其他特定于Windows的方式。

验证MySQL安装：

在MySQL已经安装成功后，基表已被初始化，服务器已经启动，可以验证所有的工作，因为它应该通过一些简单的测试。

使用mysqladmin实用程序来获取服务器状态：

使用mysqladmin检查服务器版本的二进制文件。这个二进制文件将可在Linux上的/usr/bin中或在C\ MYSQL\BIN的窗口。 

[root@host]# mysqladmin --version

在Linux上，它会产生以下结果。它可能会有所不同，具体取决于你的安装环境： 

mysqladmin  Ver 8.23 Distrib 5.0.9-0, for redhat-linux-gnu on i386

如果没有得到这样的消息，那么有可能是在安装一些问题，会需要一些帮助来解决它。

使用MySQL客户端执行简单的SQL命令：

可以使用mysql命令使用MySQL客户端连接到MySQL服务器。此时，不需要提供任何密码，因为默认情况下，它被设置为空。
因此，只要使用以下命令 


[root@host]# mysql

mysql>提示符下。现在已经连接到MySQL服务器，可以在mysql>提示符下执行SQL命令如下。 

mysql> SHOW DATABASES;
+----------+
| Database |
+----------+
| mysql    |
| test     |
+----------+
2 rows in set (0.13 sec)

安装后的步骤：

MySQL的root用户密码为空。只要已经成功安装了数据库和客户端，需要设置一个root密码如下： 

[root@host]# mysqladmin -u root password "new_password";

现在连接到MySQL服务器，使用下面的命令： 

[root@host]# mysql -u root -p
Enter password:*******

Unix用户也希望把MySQL在PATH中目录，这样就不必每次要命令行客户端输入的完整路径。对于bash可以如下： 

export PATH=$PATH:/usr/bin:/usr/sbin

在开机时运行MySQL：

如果想在系统启动时运行MySQL服务器，然后确保在/etc/rc.local文件中有以下内容： 

/etc/init.d/mysqld start

此外，应该也有一个mysqld二进制文件在/etc/init.d目录。
