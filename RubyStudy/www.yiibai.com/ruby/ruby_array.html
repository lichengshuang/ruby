Ruby数组(Array)

Ruby的数组array是有序的，任何对象的整数索引的集合。数组中的每个元素与指数。

Ruby的数组是有序的，任何对象的整数索引的集合。数组中的每个元素与指数。
数组索引从0开始，如C或Java。负指数假设数组的末尾---是，指数为-1表示最后一个元素，-2是数组中的最后一个元素，等等。
Ruby的数组能容纳的对象，如字符串，整数，Fixnum对象，哈希，符号，甚至其他的Array对象。 Ruby的数组是没有像在其他语言的阵列刚性。 Ruby的数组自动增长，同时增加他们的元素.

创建数组:

有许多方法来创建或初始化数组。一种方法是用新的类方法: 

names = Array.new

在创建阵列时，您可以设置一个数组的大小: 

names = Array.new(20)

该数组的名称现在有一个的20元素的的的的的大小或的长度。你可以返回一个数组的大小与大小或长度的方法: 

months.size  # This returns 20
months.length # This also returns 20

您可以指定阵列中的每个元素值如下: 

#!/usr/bin/ruby

names = Array.new(4, "mac")

puts "#{names}"

This will produce following result: 

macmacmacmac

您还可以使用新块，填充块评估的每个元素: 

#!/usr/bin/ruby

nums = Array.new(10) { |e| e = e * 2 }

puts "#{nums}"

这将产生以下结果: 

024681012141618

还有另一个阵列的方法，[]。它是这样工作: 

nums = Array.[](1, 2, 3, 4,5)

创建数组的形式如下 : 

nums = Array[1, 2, 3, 4,5]

内核模块的核心Ruby，有一个阵列的方法，该方法只接受一个参数。这里的方法作为参数的范围，以创建一个数字数组: 

#!/usr/bin/ruby

digits = Array(0..9)

puts "#{digits}"

这将产生以下结果: 

0123456789

数组内建方法:

我们需要有一个Array对象的实例，调用一个数组的方法。正如我们所看到的，下面是创建一个Array对象的实例: 

Array.[](...) [or] Array[...] [or] [...]

这将返回给定对象的填充一个新的数组。现在我们可以使用创建的对象调用任何可用的实例方法。例如: 

#!/usr/bin/ruby

digits = Array(0..9)

num = digits.at(6)

puts "#{num}"

这将产生以下结果: 

6

例子:

尝试下面的例子来包装各种数据. 

a = [ "a", "b", "c" ]
n = [ 65, 66, 67 ]
a.pack("A3A3A3")   #=> "a  b  c  "
a.pack("a3a3a3")   #=> "a\000\000b\000\000c\000\000"
n.pack("ccc")      #=> "ABC"
