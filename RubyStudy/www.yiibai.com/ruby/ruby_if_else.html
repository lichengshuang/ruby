Ruby if else

Ruby提供contional是很常见的现代语言的结构。在这里，我们将解释所有Ruby中条件语句和修饰符

Ruby if...else语句：

语法： 

if conditional [then]
	  code...
[elsif conditional [then]
	  code...]...
[else
	  code...]
end
如果表达式是用于条件执行。值false和nil是假的，一切是真实的。公告Ruby使用ELSIF，不是别的，如果也不ELIF。

如果条件为真执行代码。如果条件是不正确的，执行else子句中指定的代码。

一个表达的条件是从代码分离的保留字，那么，一个换行符或分号。

例如： 

#!/usr/bin/ruby

x=1
if x > 2
   puts "x is greater than 2"
elsif x <= 2 and x!=0
   puts "x is 1"
else
   puts "I can't guess the number"
end
这将输出结果如下： 
x is 1

Ruby if 修饰符：

语法： 

code if condition
如果条件为真执行代码。

例如：
#!/usr/bin/ruby

$debug=1
print "debug\n" if $debug

这将输出结果如下： 

debug

Ruby unless语句：

语法：
unless conditional [then]
   code
[else
   code ]
end
执行代码，如果有条件的话是假的。如果条件为真，执行else子句中指定的代码。

例如：
#!/usr/bin/ruby

x=1
unless x>2
   puts "x is less than 2"
 else
  puts "x is greater than 2"
end
这将输出结果如下： 
x is less than 2

Ruby unless修饰符：

语法： 
code unless conditional
执行代码，如果有条件的话是假的。

例子： 

#!/usr/bin/ruby

$var =  1
print "1 -- Value is set\n" if $var
print "2 -- Value is set\n" unless $var

$var = false
print "3 -- Value is set\n" unless $var

这将输出结果如下： 

1 -- Value is set
3 -- Value is set

Ruby case 声明:

语法： 
case expression
[when expression [, expression ...] [then]
   code ]...
[else
   code ]
end
比较的情况下由指定的表达和使用===运算符时，指定的执行代码时，比赛的第。
子句评估作为左操作数时，由指定的表达。如果没有条款匹配时，案件执行else子句中的代码。
一个语句的表达时从代码中分离出来，由保留字，一个换行符或分号。
因此： 
case expr0
when expr1, expr2
   stmt1
when expr3, expr4
   stmt2
else
   stmt3
end
基本上是类似于以下内容： 
_tmp = expr0
if expr1 === _tmp || expr2 === _tmp
   stmt1
elsif expr3 === _tmp || expr4 === _tmp
   stmt2
else
   stmt3
end
例如： 
#!/usr/bin/ruby

$age =  5
case $age
when 0 .. 2
    puts "baby"
when 3 .. 6
    puts "little child"
when 7 .. 12
    puts "child"
when 13 .. 18
    puts "youth"
else
    puts "adult"
end
这将输出结果如下： 
little child
