Ruby字符串

Ruby的​​String对象拥有和操纵的一个典型代表，代表人类语言的字符的一个或多个字节的任意序列。

Ruby的String对象拥有和操纵的一个典型代表，代表人类语言的字符的一个或多个字节的任意序列。

最简单的字符串括在单引号（撇号字符）。引号内的文字是字符串值： 

'This is a simple Ruby string literal'

如果您需要将一个单引号内的单引号的字符串文字，前面加上一个反斜杠，这样的Ruby解释器不认为它会终止字符串： 

'Won\'t you read O\'Reilly\'s book?'

反斜杠也可以逃脱另一个反斜杠，所以第二个反斜线本身并不解释为转义字符。
以下是字符串相关的Ruby特性。

表达式置换:

表达式置换是嵌入到一个使用＃{和}字符串任何Ruby表达式的值的一种手段： 

#!/usr/bin/ruby

x, y, z = 12, 36, 72
puts "The value of x is #{ x }."
puts "The sum of x and y is #{ x + y }."
puts "The average was #{ (x + y + z)/3 }."

这将产生以下结果: 

The value of x is 12.
The sum of x and y is 48.
The average was 40.

生成分隔字符串:

与一般的分隔字符串，你可以创建一对匹配的任意分隔符，例如，虽然里面的字符串（{，<等，在此之前由百分号（％）。Q，Q和X有特殊的含义一般分隔的字符串可以嵌套： 

%{Ruby is fun.}  equivalent to "Ruby is fun."
%Q{ Ruby is fun. } equivalent to " Ruby is fun. "
%q[Ruby is fun.]  equivalent to a single-quoted string
%x!ls! equivalent to back tick command output `ls`

字符编码:

为Ruby的默认字符集是ASCII，可以由单个字节表示的字符。如果你使用UTF-8，或另一种现代的字符集，字符可能会在一至四个字节表示。
你可以改变你的字符集，这样，在你的程序开始时使用$ KCODE： 

$KCODE = 'u'

以下是$ KCODE的可能值：

Code Description
a ASCII (same as none). This is the default.
e EUC.
n None (same as ASCII).
u UTF-8.

String 内建函数/方法:

我们需要有一个String对象的实例，调用String方法。以下的方式来创建String对象的一个实例： 

new [String.new(str="")]

这将返回一个新的字符串对象，其中包含str的副本。现在使用str的对象，我们可以提供任何实例方法。例如： 

#!/usr/bin/ruby

myStr = String.new("THIS IS TEST")
foo = myStr.downcase

puts "#{foo}"

这将产生以下结果： 

this is test

例子:

尝试下面的例子来解开各种数据。 

"abc \0\0abc \0\0".unpack('A6Z6')   #=> ["abc", "abc "]
"abc \0\0".unpack('a3a3')           #=> ["abc", " \000\000"]
"abc \0abc \0".unpack('Z*Z*')       #=> ["abc ", "abc "]
"aa".unpack('b8B8')                 #=> ["10000110", "01100001"]
"aaa".unpack('h2H2c')               #=> ["16", "61", 97]
"\xfe\xff\xfe\xff".unpack('sS')     #=> [-2, 65534]
"now=20is".unpack('M*')             #=> ["now is"]
"whole".unpack('xax2aX2aX1aX2a')    #=> ["h", "e", "l", "l", "o"]
