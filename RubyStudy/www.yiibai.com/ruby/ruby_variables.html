Ruby变量

变量是持有被任何程序使用的任何数据的存储位置。 有五种类型的变量由Ruby支持。你已经走在前面的章节，以及这些变量通过小的描述。本章解释这五种类型的变量。
变量是持有被任何程序使用的任何数据的存储位置。
有五种类型的变量由Ruby支持。你已经走在前面的章节，以及这些变量通过小的描述。本章解释这五种类型的变量。
 

Ruby全局变量：


全局变量以$开头。未初始化的全局变量有值为零，与-w选项产生警告。
分配给全局变量改变全球的地位。这是不建议使用全局变量。他们做节目神秘。
下面是一个例子显示使用全局变量。
#!/usr/bin/ruby

$global_variable = 10
class Class1
  def print_global
     puts "Global variable in Class1 is #$global_variable"
  end
end
class Class2
  def print_global
     puts "Global variable in Class2 is #$global_variable"
  end
end

class1obj = Class1.new
class1obj.print_global
class2obj = Class2.new
class2obj.print_global
这里$ global_variable是一个全局变量。这将产生以下结果：
注意：在Ruby中，你可以访问任何价值把之前的井号（＃）字符变量或常量的变量或常量。 
Global variable in Class1 is 10
Global variable in Class2 is 10

Ruby的实例变量：


实例变量以@开始。初始化实例变量的值为零，与-w选项产生警告。
这里是一个例子显示使用实例变量。

#!/usr/bin/ruby

class Customer
   def initialize(id, name, addr)
      @cust_id=id
      @cust_name=name
      @cust_addr=addr
   end
   def display_details()
      puts "Customer id #@cust_id"
      puts "Customer name #@cust_name"
      puts "Customer address #@cust_addr"
    end
end

# Create Objects
cust1=Customer.new("1", "John", "Wisdom Apartments, Ludhiya")
cust2=Customer.new("2", "Poul", "New Empire road, Khandala")

# Call Methods
cust1.display_details()
cust2.display_details()
这里@cust_id，cust_name和@cust_addr的实例变量。这将产生以下结果：
Customer id 1
Customer name John
Customer address Wisdom Apartments, Ludhiya
Customer id 2
Customer name Poul
Customer address New Empire road, Khandala

Ruby类的变量：

类变量以@@ 开始，必须初始化之前，他们可以在方法定义中使用，。
引用一个未初始化的类变量产生一个错误。类变量之间共享的类或模块中的类变量定义的后裔。
重写类变量与-w选项产生的警告。
下面是一个例子显示使用类变量：
#!/usr/bin/ruby

class Customer
   @@no_of_customers=0
   def initialize(id, name, addr)
      @cust_id=id
      @cust_name=name
      @cust_addr=addr
   end
   def display_details()
      puts "Customer id #@cust_id"
      puts "Customer name #@cust_name"
      puts "Customer address #@cust_addr"
    end
    def total_no_of_customers()
       @@no_of_customers += 1
       puts "Total number of customers: #@@no_of_customers"
    end
end

# Create Objects
cust1=Customer.new("1", "John", "Wisdom Apartments, Ludhiya")
cust2=Customer.new("2", "Poul", "New Empire road, Khandala")

# Call Methods
cust1.total_no_of_customers()
cust2.total_no_of_customers()
在这里@@no_of_customers是一个类变量。这将产生以下结果：
Total number of customers: 1
Total number of customers: 2
Ruby本地变量：

局部变量用小写字母或_开始。从class，module，def，或做相应的结束，或从一个块的左大括号密切括号{}局部变量范围的范围。

当被引用一个未初始化的局部变量，它被解释为一个没有参数的方法调用。

分配给未初始化的局部变量也作为变量声明。变数开始存在，直到达到当前范围的结尾。局部变量的生命周期的Ruby解析程序时确定。

Ruby的常量：


常量以大写字母开始。在类或模块中定义的常量可以访问内，类或模块，类或模块定义外，可全局访问。
常量不得在方法定义。引用一个未初始化的常数产生一个错误。分配到一个已经初始化的常数会产生一个警告。
#!/usr/bin/ruby

class Example
   VAR1 = 100
   VAR2 = 200
   def show
       puts "Value of first Constant is #{VAR1}"
       puts "Value of second Constant is #{VAR2}"
   end
end

# Create Objects
object=Example.new()
object.show
这里VAR1和VAR2是不变的。这将产生以下结果：
Value of first Constant is 100
Value of second Constant is 200

  •     false：代表虚假的价值。
     
  •     nil：代表未定义的值。

Ruby基本文字：


Ruby对文字的使用规则是简单和直观。本节介绍了所有基本的Ruby文字。

整数：

Ruby支持整数。一个整数的范围可以从-230到230-1或-262至262-1。存储在类BIGNUM对象中，与在此范围内的整数是类Fixnum超出这个范围的整数对象。

你写的整数用一个可选的前导符号，一个可选的酸碱指示剂（0八进制，为0x十六进制，或者二进制0B），由串数字，在适当的基础。下划线字符将被忽略的数字串。

你还可以得到整数值对应的ASCII字符或转义序列，由它前面加上一个问号。

例如：
123                  # Fixnum decimal
1_234                # Fixnum decimal with underline
-500                 # Negative Fixnum
0377                 # octal
0xff                 # hexadecimal
0b1011               # binary
?a                   # character code for 'a'
?\n                  # code for a newline (0x0a)
12345678901234567890 # Bignum
注：类和对象是在本教程中的单独一章解释。
 

浮点数：


Ruby支持整数。他们还电话号码，但小数。浮点数Float类的对象，可以是下列任何一项：

例如：
123.4                # floating point value
1.0e6                # scientific notation
4E20                 # dot not required
4e+20                # sign before exponential

字符串文字：


Ruby字符串是简单的8位字节序列，他们是String类的对象。双引号字符串可以替代和反斜线符号，但不允许单引号替换，并允许反斜线符号仅用于\ \\'

例如：
#!/usr/bin/ruby -w

puts 'escape using "\\"';
puts 'That\'s right';
这将输出以下结果：
escape using "\"
That's right
您也可以替换成一个字符串，使用序列＃{expr}的任何Ruby表达式的值。在这里，expr的可能是任何Ruby的表达式。 
#!/usr/bin/ruby -w

puts "Multiplication Value : #{24*60*60}";
这将输出以下结果：
Multiplication Value : 86400

Ruby的数组：


创建对象引用方括号内放置一个逗号分隔的一系列Ruby数组的文字。一个尾随逗号被忽略。
#!/usr/bin/ruby

ary = [  "fred", 10, 3.14, "This is a string", "last element", ]
ary.each do |i|
   puts i
end
这将输出结果如下：
fred
10
3.14
This is a string
last element

Ruby 哈希：


文字的Ruby创建哈希放在括号之间的键/值对的列表，用逗号或=>之间的键和值序列。一个尾随逗号被忽略。

例如： 

#!/usr/bin/ruby

hsh = colors = { "red" => 0xf00, "green" => 0x0f0, "blue" => 0x00f }
hsh.each do |key, value|
   print key, " is ", value, "\n"
end

这将输出结果如下： 

green is 240
blue is 15
red is 3840

如需详细Ruby哈希，通过Ruby哈希了解更多。


Ruby的Range：


一个范围代表一个开始和结束值interval.a集。范围可使用.. E和...é文字，或Range.new与构建。
范围使用......包含从开始运行到结束。那些使用...排除最终价值。作为一个迭代器使用时，范围返回序列中的每个值。
一个范围（1 .. 5）意味着它包括1，2，3，4，5个值和范围（1 ... 5），这意味着它包括1，2，3，4个值。
例如：
#!/usr/bin/ruby

(10..15).each do |n| 
   print n, ' ' 
end
这将输出结果如下：
10 11 12 13 14 15
