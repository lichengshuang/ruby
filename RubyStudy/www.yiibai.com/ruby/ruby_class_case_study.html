Ruby类实例

Ruby类和对象学习，您将创建一个自定义的Ruby类，您将声明两个方法。
Ruby类案例研究，您将创建一个叫Customer的Ruby类，您将声明两个方法。
  • display_details：方法将显示客户的详细信息。
  • total_no_of_customers：方法将显示在系统中创建的客户总数。 
#!/usr/bin/ruby

class Customer
   @@no_of_customers=0
   def initialize(id, name, addr)
      @cust_id=id
      @cust_name=name
      @cust_addr=addr
   end
   def display_details()
      puts "Customer id #@cust_id"
      puts "Customer name #@cust_name"
      puts "Customer address #@cust_addr"
    end
    def total_no_of_customers()
       @@no_of_customers += 1
       puts "Total number of customers: #@@no_of_customers"
    end
end
display_details方法包含三个把报表显示客户ID，客户名称，客户地址。 puts语句：
puts "Customer id #@cust_id"
将显示文本的客户ID值的变量@单行CUST_ID。

当你想在单行显示的文字和实例变量的值，你需要在变量名前面puts语句的hash符号（＃）。随着哈希符号（＃）的文字和实例变量应该被括在双引号。

第二种方法，total_no_of_customers，是一种方法，它包含了类变量@ @ no_of_customers。表达@ @ no_of_客户+= 1加1的方法total_no_of_customers每次被称为变量no_of_customers。这样，你将永远有客户总数中的类变量。

现在创建两个客户如下：
cust1=Customer.new("1", "John", "Wisdom Apartments, Ludhiya")
cust2=Customer.new("2", "Poul", "New Empire road, Khandala")
在这里，我们创建作为cust1和cust2的Customer类的两个对象，并通过必要的参数，用新的方法。 initialize方法被调用，并初始化对象的必要属性。

一旦对象被创建，你需要调用类的方法，通过使用两个对象。如果你想调用一个方法或数据成员，你写以下内容：
cust1.display_details()
cust1.total_no_of_customers()
对象的名称应始终遵循一个点，这反过来又其次是方法的名称或任何数据成员。我们已经看到了如何调用这两种方法，由使用cust1对象。使用cust2的对象，你可以调用两种方法如下所示：
cust2.display_details()
cust2.total_no_of_customers()

#!/usr/bin/ruby

class Customer
   @@no_of_customers=0
   def initialize(id, name, addr)
      @cust_id=id
      @cust_name=name
      @cust_addr=addr
   end
   def display_details()
      puts "Customer id #@cust_id"
      puts "Customer name #@cust_name"
      puts "Customer address #@cust_addr"
   end
   def total_no_of_customers()
      @@no_of_customers += 1
      puts "Total number of customers: #@@no_of_customers"
   end
end

# Create Objects
cust1=Customer.new("1", "John", "Wisdom Apartments, Ludhiya")
cust2=Customer.new("2", "Poul", "New Empire road, Khandala")

# Call Methods
cust1.display_details()
cust1.total_no_of_customers()
cust2.display_details()
cust2.total_no_of_customers()
现在运行这个程序如下：
$ ruby cust.rb
这将产生以下结果：
Customer id 1
Customer name John
Customer address Wisdom Apartments, Ludhiya
Total number of customers: 1
Customer id 2
Customer name Poul
Customer address New Empire road, Khandala
Total number of customers: 2
