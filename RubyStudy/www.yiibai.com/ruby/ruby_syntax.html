Ruby语法

让我们写一个简单的ruby程序。所有的Ruby文件将具有扩展.rb。下面的源代码所以把在test.rb文件的。
让我们写一个简单的ruby程序。所有的Ruby文件将具有扩展.rb。下面的源代码所以把在test.rb文件的。
#!/usr/bin/ruby -w

puts "Hello, Ruby!";
在这里，我假设你有/usr/bin目录中提供的Ruby解释。现在尝试运行这个程序如下：
$ ruby test.rb
这将输出结果如下：
Hello, Ruby!
您已经看到了一个简单的Ruby程序，现在让我们看看Ruby语法有关的几个基本概念：
 

Ruby程序中的空白：


在Ruby代码中的空白字符，如空格和制表符一般被忽略，除非当他们出现在字符串中。然而，有时候，他们用来解释模棱两可的语句。 -w选项启用时，这种解释产生警告。

例如：
a + b is interpreted as a+b ( Here a is a local variable)
a  +b is interpreted as a(+b) ( Here a is a method call)

Ruby程序中的行结尾：


Ruby的解释分号作为语句的结尾换行符。但是，如果红宝石遭遇运营商，如+， - ，或在一行的末尾的反斜杠，他们的一份声明中表示的延续。
 

Ruby的标识符：


标识符名称的变量，常量和方法。 Ruby的标识符是大小写敏感。这意味着RAM和RAM是在Ruby中的两个不同的itendifiers。
Ruby的标识符名称可以包含字母数字字符和下划线（_）。
 

保留字：


以下列表显示了在Ruby中的保留字。作为常量或变量的名称，不得使用这些保留字。然而，他们可以被用来作为方法名称。
 
 
BEGIN do next then
END else nill true
alias elsif not undef
and end or unless
begin ensure redo until
break false rescue when
case for retry while
class if return while
def in self __FILE__
defined? module super __LINE__

Ruby的“Here Document”：


"Here Document"是指建立多行字符串。 <<你可以指定一个字符串或的标识符终止字符串文字，终结后，当前行的所有行是字符串值。
如果终结者被引用，引号的类型确定的面向行的字符串文字的类型。注意必须有<<和终止之间没有空格。
这里有不同的例子：
#!/usr/bin/ruby -w

print <<EOF
    This is the first way of creating
    here document ie. multiple line string.
EOF

print <<"EOF";                # same as above
    This is the second way of creating
    here document ie. multiple line string.
EOF

print <<`EOC`                 # execute commands
	echo hi there
	echo lo there
EOC

print <<"foo", <<"bar"  # you can stack them
	I said foo.
foo
	I said bar.
bar
这将输出结果如下：
This is the first way of creating
    her document ie. multiple line string.
    This is the second way of creating
    her document ie. multiple line string.
hi there
lo there
        I said foo.
        I said bar.


Ruby的Begin语句：


语法：
BEGIN {
   code
}

声明将在程序的开始调用的代码。

例子：

#!/usr/bin/ruby

puts "This is main Ruby Program"

BEGIN {
   puts "Initializing Ruby Program"
}

这将输出结果如下：

Initializing Ruby Program
This is main Ruby Program

Ruby的End语句：


语法：
END {
   code
}
声明将在程序的最后调用的代码。

例子：
#!/usr/bin/ruby

puts "This is main Ruby Program"

END {
   puts "Terminating Ruby Program"
}
BEGIN {
   puts "Initializing Ruby Program"
}
这将输出结果如下：
Initializing Ruby Program
This is main Ruby Program
Terminating Ruby Program


Ruby的注释：


注释隐藏一条线，一条线的一部分，或从Ruby解释器的几行。在一行的开头，您可以使用散列符（＃）
# I am a comment. Just ignore me.
或注释可能会后的声明或表达同一行
name = "Madisetti" # This is again comment
你可以注释多行如下：
# This is a comment.
# This is a comment, too.
# This is a comment, too.
# I said that already.
这里是另一种形式。此块注释隐瞒几行解释=begin/=end：
=begin
This is a comment.
This is a comment, too.
This is a comment, too.
I said that already.
=end
