Ruby循环

在Ruby中的循环使用执行相同的代码块指定的次数。本章详细介绍了由Ruby支持的所有的循环语句while,for,until。

在Ruby中的循环使用执行相同的代码块指定的次数。本章详细介绍了由Ruby支持的所有的循环语句while,for,until。

Ruby while声明：

语法： 

while conditional [do]
   code
end
执行while的条件是真实的的的的代码。一个while循环的条件是保留字做，换行，反斜杠\，或分号分隔的代码;

例如： 

#!/usr/bin/ruby

$i = 0;
$num = 5;

while $i < $num  do
   puts("Inside the loop i = #$i" );
   $i +=1;
end
这将输出结果如下： 
Inside the loop i = 0
Inside the loop i = 1
Inside the loop i = 2
Inside the loop i = 3
Inside the loop i = 4

Ruby while修饰符：

语法： 

code while condition

OR

begin 
  code 
end while conditional
执行while的条件是真实的的的的代码。

而修饰符如下如果开始没有救援的声明或保证条款，一旦代码被执行之前，有条件的进行评估。
例如：
#!/usr/bin/ruby

$i = 0;
$num = 5;
begin
   puts("Inside the loop i = #$i" );
   $i +=1;
end while $i < $num
这将输出以下结果： 
Inside the loop i = 0
Inside the loop i = 1
Inside the loop i = 2
Inside the loop i = 3
Inside the loop i = 4

Ruby until 语句：

语法： 

until conditional [do]
   code
end

执行代码，而条件是虚假的。直到语句的条件是分开的代码保留字做一个换行符或分号。
例如： 

#!/usr/bin/ruby

$i = 0;
$num = 5;

until $i > $num  do
   puts("Inside the loop i = #$i" );
   $i +=1;
end
这将产生以下结果: 
Inside the loop i = 0
Inside the loop i = 1
Inside the loop i = 2
Inside the loop i = 3
Inside the loop i = 4
Inside the loop i = 5

Ruby until 修饰符：

语法： 

code until conditional

OR

begin
   code
end until conditional
执行代码，而条件是虚假的。
如果直到修饰符如下开始没有救援的声明或保证条款，一旦代码被执行之前，有条件进行评估。

例如： 
#!/usr/bin/ruby

$i = 0;
$num = 5;
begin
   puts("Inside the loop i = #$i" );

这将输出以下结果： 

Inside the loop i = 0
Inside the loop i = 1

Ruby for 语句：

语法： 

for variable [, variable ...] in expression [do]
   code
end
一次执行代码表达式中的每个元素。

例如：
#!/usr/bin/ruby

for i in 0..5
   puts "Value of local variable is #{i}"
end
在这里，我们定义范围为0.. 5。 0 ..5我在发言将使我要在范围值从0到5（含5人），这将产生以下结果： 
Value of local variable is 0
Value of local variable is 1
Value of local variable is 2
Value of local variable is 3
Value of local variable is 4
Value of local variable is 5
一个循环中......几乎是完全等同于： 
(expression).each do |variable[, variable...]| code end
除了一个循环不创建一个新的局部变量的范围。从代码分离为一个循环的表达保留字，一个换行符或分号。

例如：
#!/usr/bin/ruby

(0..5).each do |i|
   puts "Value of local variable is #{i}"
end
这将输出以下结果： 
Value of local variable is 0
Value of local variable is 1
Value of local variable is 2
Value of local variable is 3
Value of local variable is 4
Value of local variable is 5

Ruby break 语句：

语法： 

break
终止的最内部的循环。终止与相关块的方法，如果在调用块（方法返回nil）。
例如：
#!/usr/bin/ruby

for i in 0..5
   if i > 2 then
      break
   end
   puts "Value of local variable is #{i}"
end
这将输出以下结果： 
Value of local variable is 0
Value of local variable is 1
Value of local variable is 2

Ruby next 语句：

语法： 

next
跳转到下一次迭代的最内部的循环。终止执行一个块，如果在一个块内调用（产量或拨打返回nil）。

例如：
#!/usr/bin/ruby

for i in 0..5
   if i < 2 then
      next
   end
   puts "Value of local variable is #{i}"
end
这将输出以下结果： 
Value of local variable is 2
Value of local variable is 3
Value of local variable is 4
Value of local variable is 5

Ruby redo 语句：

语法： 

redo
重新启动这个最内部的循环迭代，而不检查循环条件。重新启动产生或致电如果一个块内调用。
例如：
#!/usr/bin/ruby

for i in 0..5
   if i < 2 then
      puts "Value of local variable is #{i}"
      redo
   end
end
这将产生以下结果，并会在一个无限循环中去： 
Value of local variable is 0
Value of local variable is 0
............................

Ruby retry 语句：

语法： 

retry
如果重试出现在救援开始表达条，从的1begin机构开始重新启动。 
begin
   do_something # exception raised
rescue
   # handles error
   retry  # restart from beginning
end
如果重试出现在迭代器块，或身体的表达，重新启动调用迭代器调用。迭代器的参数被重新评估。 
for i in 1..5
   retry if some_condition # restart from i == 1
end
例如： 
#!/usr/bin/ruby

for i in 1..5
   retry if  i > 2
   puts "Value of local variable is #{i}"
end
这将产生以下结果，并会在一个无限循环中去： 
Value of local variable is 1
Value of local variable is 2
Value of local variable is 1
Value of local variable is 2
Value of local variable is 1
Value of local variable is 2
............................
