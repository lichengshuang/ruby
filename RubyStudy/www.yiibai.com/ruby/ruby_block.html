你已经看到Ruby如何定义的方法，你可以把一些语句然后调用该方法。同样地，Ruby的Bolck的概念。
  • 一个块包含的代码块。
  • 您分配一个名称，一个块。
  • 块中的代码总是被括在大括号（{}）。
  • 块与块的名称相同的函数调用。这意味着，如果你有一个名称测试块，那么你使用的功能测试，调用这个块。
  • 你调用使用yield语句块。

语法： 

block_name{
   statement1
   statement2
   ..........
}

在这里，您将学习如何通过使用一个简单的yield语句调用块。您还将学习到yield语句使用参数调用块。您将检查与yield语句的两种类型的示例代码。

yield语句：

让我们看在yield语句的例子： 
#!/usr/bin/ruby

def test
   puts "You are in the method"
   yield
   puts "You are again back to the method"
   yield
end
test {puts "You are in the block"}
这将输出结果如下： 
You are in the method
You are in the block
You are again back to the method
You are in the block
你也可以通过yield语句的参数。下面是一个例子： 
#!/usr/bin/ruby

def test
   yield 5
   puts "You are in the method test"
   yield 100
end
test {|i| puts "You are in the block #{i}"}
这将输出结果如下： 
You are in the block 5
You are in the method test
You are in the block 100


这里yield语句被写入随后的参数。你甚至可以传递多个参数。块中，您将两条垂直线之间的变量（| |）接受的参数。因此，在前面的代码，产量5日发表声明，通过试块的参数值5。
现在看看下面的语句： 

test {|i| puts "You are in the block #{i}"}
这里收到变量i的值5。现在观察提出声明如下： 
puts "You are in the block #{i}"
这使得语句的输出是： 
You are in the block 5
如果你想传递多个参数，然后yield语句变成： 
yield a, b
块是： 
test {|a, b| statement}


将用逗号隔开的参数。

块和方法：

你已经看到如何如何块和方法可以与每个other.You正常调用的方法具有相同的名称作为该块使用yield语句块。因此，你写的： 
def test
  yield
end
test{ puts "Hello world"}
这个例子是最简单的方式来实现使用yield语句测试块。
但如果是由与之前的最后一个参数的方法，那么您可以通过此方法块，该块将被分配到最后一个参数。的情况下两个*和＆参数列表中，及应晚一点。 
def test(&block)
   block.call
end
test { puts "Hello World!"}
这将产生以下结果： 
Hello World!

BEGIN和END块

每个Ruby源文件可以声明要运行的文件正在加载（BEGIN块）后，该计划已执行完毕（END块）的代码块。 
BEGIN { 
   begin block code 
} 

END { 
   end block code 
}
一个程序可能包含多个BEGIN和END块。在他们遇到的顺序执行BEGIN块， END块执行顺序相反。
