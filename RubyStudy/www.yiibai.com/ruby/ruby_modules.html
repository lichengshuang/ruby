Ruby模块

Ruby模块(module)是组合在一起的方法，类和常量的方式。模块为您提供两大好处。

模块是组合在一起的方法，类和常量的方式。模块为您提供两大好处。

  • 模块提供了一个命名空间，并防止名称冲突。
  • 模块实现混入设施。
模块定义一个命名空间，一个沙箱中，可以发挥你的方法和常量，而不必担心正在加紧通过其他方法和常量。

语法: 

module Identifier
   statement1
   statement2
   ...........
end

被命名为模块中的常量，就像类中的常量，与最初的大写字母。该方法的定义看起来很相似，也就像类的方法定义模块的方法。
作为类的方法，你打电话的module.s名称和一个时期它的名字前面加上一个模块方法，并引用一个常数，使用模块名和两个冒号。

例子: 

#!/usr/bin/ruby

# Module defined in trig.rb file

module Trig
   PI = 3.141592654
   def Trig.sin(x)
   # ..
   end
   def Trig.cos(x)
   # ..
   end
end

我们可以定义一个函数的名称相同，但不同的功能模块： 

#!/usr/bin/ruby

# Module defined in moral.rb file

module Moral
   VERY_BAD = 0
   BAD = 1
   def Moral.sin(badness)
   # ...
   end
end

类的方法一样，当你在一个模块中定义一个方法，您可以指定一个点模块名称和方法名称。

Ruby require 语句:

需要声明的是类似的C的include语句的C ++和Java的import语句。第三个程序如果要使用任何已定义的模块，它可以简单地使用Ruby的require语句加载的模块文件：

语法: 

require filename

这里是不是需要给文件名扩展.rb。

例子: 

require 'trig.rb'
require 'moral'

y = Trig.sin(Trig::PI/4)
wrongdoing = Moral.sin(Moral::VERY_BAD)

重要提示: 两个文件都包含相同的函数名。因此，这将导致代码中的含糊之处，而包括在调用程序，但模块避免此代码模糊，我们都可以调用相应的功能，使用模块名称。

Ruby include 语句:

你可以在一个类中嵌入一个模块。嵌入到一个类模块，您可以使用类includestatement：

语法: 

include modulename

如果一个模块是单独的文件中定义，那么它必须包括该文件之前，使用的重构，在类模块requirestatement.

例子:

考虑下面的模块在Week.rb文件。  

module Week
   FIRST_DAY = "Sunday"
   def Week.weeks_in_month
      puts "You have four weeks in a month"
   end
   def Week.weeks_in_year
      puts "You have 52 weeks in a year"
   end
end

现在您可以包括在这个模块在一个类里，如下： 

#!/usr/bin/ruby
require "Week"

class Decade
include Week
   no_of_yrs=10
   def no_of_months
      puts Week::FIRST_DAY
      number=10*12
      puts number
   end
end
d1=Decade.new
puts Week::FIRST_DAY
Week.weeks_in_month
Week.weeks_in_year
d1.no_of_months

这将产生以下结果： 

Sunday
You have four weeks in a month
You have 52 weeks in a year
Sunday
120

Ruby中Mixins:

通过本节之前，我假定你有知识的面向对象的概念。
当一个类可以继承多个父类的功能，这个类应该显示多重继承。
Ruby没有多重继承直接支持，但Ruby模块，美妙的使用。他们几乎消除了多重继承的需要，提供了一个工厂，一个mixin。
混入给你的奇妙的控制方式，增加类的功能。然而，他们的真正的力量出来时，代码中混入开始使用它的类中的代码进行交互。
让我们来看看下面的示例代码，以获得一个mixin的理解： 


module A
   def a1
   end
   def a2
   end
end
module B
   def b1
   end
   def b2
   end
end

class Sample
include A
include B
   def s1
   end
end

samp=Sample.new
samp.a1
samp.a2
samp.b1
samp.b2
samp.s1

模块A由A1和A2的方法。模块B包括B1和B2的方法。类的样本包括两个模块A和B类样品可以访问所有四种方法，即A1，A2，B1，B2。因此，你可以看到，从模块继承类样品。因此，可以说类的示例显示了多重继承或一个mixin。
