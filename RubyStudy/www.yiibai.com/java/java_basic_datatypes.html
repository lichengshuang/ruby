Java基础数据类型

Java基础数据类型编程教程,例子教程,参考手册和代码-基于变量的数据类型，操作系统分配内存，并决定在保留内存可以存储。

变量是什么，但保留存储器位置来存储值。这意味着，当你创建一个变量，你在内存中预留一些空间.

基于变量的数据类型，操作系统分配内存，并决定在保留内存可以存储。因此，通过分配不同的数据类型变量，这些变量可以存储整数，小数，或字符。
在Java有两种数据类型:

  1. 原始数据类型

  2. 引用/对象数据类型

原始数据类型:

有8个原始数据，由Java支持的类型。原始数据类型是预定义的语言和一个关键的字命名。现在让我们来看看有关的八个原始数据类型的详细.

byte:

  • 字节的数据类型是签署一个8位二进制补码整数.

  • 最小值是 -128 (-2^7)

  • 最大值是 127 (inclusive)(2^7 -1)

  • 默认值是 0

  • 字节的数据类型用于保存在大型阵列的空间，主要是在的整数地方，因为一个字节是比int小的四倍.

  • 例子 : byte a = 100 , byte b = -50

short:

  • 短整型(short)数据类型是16位有符号2的补码整数.

  • 最小值是 -32,768 (-2^15)

  • 最大值是 32,767(inclusive) (2^15 -1)

  • 短整型(short)数据类型也可以用来保存内存字节的数据类型，比int小的2倍

  • 默认值是 0.

  • 例子: short s= 10000 , short r = -20000

int:

  • int数据类型是一个32位签署两个补码整数.

  • 最小值是 - 2,147,483,648.(-2^31)

  • 最大值是 2,147,483,647(inclusive).(2^31 -1)

  • int是通常用来作为整数值的默认数据类型，除非有一个关于内存的关注.

  • 默认值是 0.

  • 例子: int a = 100000, int b = -200000

long:

  • Long数据类型是64位有符号2的补码整数.

  • 最小值是 -9,223,372,036,854,775,808.(-2^63)

  • 最大值是 9,223,372,036,854,775,807 (inclusive). (2^63 -1)

  • 这需要更广泛的范围比int类型时使用.

  • 默认值是 0L.

  • 例子: int a = 100000L, int b = -200000L

float:

  • float数据类型是单精度32位IEEE754浮点.

  • 浮动主要是用来保存在浮点数的大型阵列的存储器.

  • 默认值是 0.0f.

  • float数据类型是从未使用过的精确值，如货币.

  • 例子 : float f1 = 234.5f

double:

  • 双数据类型是双精度64位IEEE754浮点.

  • 此数据类型通常被用作默认的数据类型为十进制值。一般默认选择.

  • Double数据类型不应该被用于精确值，如货币.

  • 默认值是 0.0d.

  • 例子 : double d1 = 123.4

boolean:

  • 布尔数据类型代表一个比特的信息.

  • 只有两个可能的值：true和false.

  • 此数据类型用于跟踪 true/false条件的简单标志.

  • 默认值是 false.

  • 例子 : boolean one = true

char:

  • char数据类型是一个16位的Unicode字符.

  • 最小值是 '\u0000' (or 0).

  • 最大值是 '\uffff' (or 65,535 inclusive).

  • char数据类型用于存储任何字符.

  • 例子 . char letterA ='A'

参考引用数据类型:

  • 引用变量使用的类定义的构造函数创建的。它们被用来访问对象。这些变量的声明是不能改变的一个特定类型。例如，员工，小狗等.

  • 类的对象，不同类型的数组变量来引用数据类型之下.

  • 任何引用变量的默认值是null.

  • 一个引用变量可以用来指任何对象声明的类型或任何兼容型.

  • 例子 : Animal animal = new Animal("giraffe");

Java的常值:

常值是一个固定值的源代码表示。他们表示，直接在没有任何计算的代码。
常值可以被分配到任何原始类型的变量。例如: 

byte a = 68;
char a = 'A'

字节，整数，长，短，可以表示十进制（基数为10），十六进制（基数为16）或八进制（基数为8），以及数字系统。
前缀0是用来表示八进制，前缀0x表示十六进制时使用这些文字系统。例如: 

int decimal = 100;
int octal = 0144;
int hexa =  0x64;

在Java中的字符串常量指定，就像他们在其他大多数语言是封闭的一对双引号之间的字符序列。字符串常量的例子: 

"Hello World"
"two\nlines"
"\"This is in quotes\""

String和char类型的文字可以包含任何Unicode字符。例如: 

char a = '\u0001';
String a = "\u0001";

Java语言支持为String和char文字，以及一些特殊的转义序列。他们是:

Notation Character represented
\n Newline (0x0a)
\r Carriage return (0x0d)
\f Formfeed (0x0c)
\b Backspace (0x08)
\s Space (0x20)
\t tab
\" Double quote
\' Single quote
\\ backslash
\ddd Octal character (ddd)
\uxxxx Hexadecimal UNICODE character (xxxx)
