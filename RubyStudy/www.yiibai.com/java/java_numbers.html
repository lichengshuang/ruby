Java Number类

Java Number类 - 通常情况下，当我们与数字打交道，我们使用原始数据类型，如字节，整型，长，双（Integer, Long, Byte, Double, Float, Short）等。

通常情况下，当我们与数字打交道，我们使用原始数据类型，如字节，整型，长，双等

实例: 

int i = 5000;
float gpa = 13.65;
byte mask = 0xaf;

然而，在发展中我们遇到的情况，我们需要使用的对象，而不是原始数据类型。为了实现这个Java为每个原始数据类型的包装类。

所有包装类（Integer, Long, Byte, Double, Float, Short）是抽象类的子类数。


这种包装是照顾的编译器，这个过程被称为装箱。因此，当原始的对象时，需要使用编译器盒的原始类型的包装类。同样地，编译器，以及一个原始unboxes对象。Number是java.lang包中的一部分。

boxing和unboxing下面是一个例子： 

public class Test{
   public static void main(String args[]){
      Integer x = 5; // boxes int to an Integer object
      x =  x + 10;   // unboxes the Integer to a int
      System.out.println(x); 
   }
}

这将产生以下结果： 

15

When x is assigned integer values, the compiler boxes the integer because x is integer objects. Later, x is unboxed so that they can be added as integers.
