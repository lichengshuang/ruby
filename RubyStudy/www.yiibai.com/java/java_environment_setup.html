Java环境设置

Java环境设置-之前我们进一步建立了正确的Java环境

之前我们进一步建立了正确的Java环境. 本节将指导您如何下载和设置您的计算机上的Java。请遵循以下步骤来设置环境.

Java SE的是免费提供的链接下载Java。所以你根据您的操作系统的版本下载.

按照说明下载Java和运行.exe安装Java到您的计算机上。一旦你的机器上安装Java，你就需要设置环境变量指向正确安装目录:

用于Windows 2000/XP的设置路径:

假设您已经安装了Java在 c:\Program Files\java\jdk directory:

  • 右键单击“我的电脑”，选择“属性”.

  • “高级”标签下点击“环境变量”按钮.

  • 现在改变的“路径”变量，因此，它也包含Java可执行文件的路径。例如，如果路径是当前设置为  'C:\WINDOWS\SYSTEM32', 然后改变你的路径为：'C:\WINDOWS\SYSTEM32;c:\Program Files\java\jdk\bin'（粗体部分为java安装路径）.

设置Windows 95/98/Me中的路径:

假设你已经安装java在 c:\Program Files\java\jdk 目录:

  • 编辑'C:\autoexec.bat' 把以下行加到文件尾部: 
    'SET PATH=%PATH%;C:\Program Files\java\jdk\bin'

设置用于在Linux，UNIX，Solaris和FreeBSD的路径:

应设置环境变量PATH来指向Java二进制文件已经安装。如果你有这样的麻烦，请参考你的shell文件.

例如，如果你使用你的shell的bash，那么你会添加下面一行到 '.bashrc: export PATH=/path/to/java:$PATH' 文件

流行的Java编辑器:

写Java程序，你需要一个文本编辑器。有在市场上更复杂的IDE。但现在，你可以考虑以下:

  • Notepad : Windows机器上，你可以使用任何简单的文本编辑器如记事本（本教程推荐），TextPad.

  • Netbeans : 是一个Java IDE，它是开源和免费，其中可以从http://www.netbeans.org/index.html下载。

  • Eclipse :也是一个Java IDE由Eclipse开源社区开发和可以从 http://www.eclipse.org/ 下载.
