Java变量类型

Java变量类型编程教程,例子教程,参考手册和代码-在Java中，所有的变量必须声明才可以使用。这里显示的变量声明的基本形式。

在Java中，所有的变量必须声明才可以使用。这里显示的变量声明的基本形式: 

type identifier [ = value][, identifier [= value] ...] ;

该类型是Java的数据类型之一。标识符是变量的名称。要声明多个指定类型的变量，使用逗号分隔的列表.

这里有几个不同类型的变量声明的例子。需要注意的是一些包括初始化. 

int a, b, c;         // declares three ints, a, b, and c.
int d = 3, e, f = 5; // declares three more ints, initializing
                     // d and f.
// by www.yiibai.com
byte z = 22;         // initializes z.
double pi = 3.14159; // declares an approximation of pi.
char x = 'x';        // the variable x has the value 'x'.

本章将介绍Java语言中的各种变量类型。有三种在Java中的变量:

  1. 局部变量

  2. 实例变量

  3. 类/静态变量

局部变量 :

  • 局部变量声明的方法，构造函数或块.

  • 进入方法时创建局部变量，一旦退出方法，构造函数或块变量将被销毁.

  • 访问修饰符不能用于局部变量.

  • 局部变量是可见的，只在声明的方法，构造函数或块.

  • 实施局部变量是在栈级别内部.

  • 没有局部变量的默认值，因此应声明和局部变量的初始值第一次使用前应分配.

实例:

这里age是一个局部变量。仅这里面pupAge()方法定义，其范围仅限于这种方法 

public class Test{ 
   public void pupAge(){
      int age = 0;
      age = age + 7;
      System.out.println("Puppy age is : " + age)
   }
   
   public static void main(String args[]){
      Test test = new Test();
      Test.pupAge();
   }
}

这将产生以下结果: 

Puppy age is: 7

实例:

下面的示例使用没有初始化age变量，所以它会给一个错误，在编译的时候. 

public class Test{ 
   public void pupAge(){
      int age;
      age = age + 7;
      System.out.println("Puppy age is : " + age)
   }
   
   public static void main(String args[]){
      Test test = new Test();
      Test.pupAge();
   }
}

这将产生以下错误，在编译时: 

Test.java:4:variable number might not have been initialized
age = age + 7;
         ^
1 error

实例变量 :

  • 实例变量可以声明在类，但外面的方法，构造或者任何块.

  • 当空间被分配在堆中的一个对象，每个实例变量的值插槽创建.

  • 当一个对象被创建的关键'new'字的使用和销毁的对象被销毁时创建实例变量.

  • 必须由一种以上的方法，构造函数或块，或一个对象状态的重要组成部分，必须出现了类通过引用实例变量持有的值.

  • 在类级前或使用后，可以声明实例变量.

  • 访问修饰符可以给实例变量.

  • 实例变量是在类的所有方法，构造和块可见。一般而言，这是建议，以使这些变量的私人访问级别。然而能见度为子类，这些变量可以使用访问修饰符.

  • 实例变量的默认值。对于数字的默认值是0，布尔值，它是false，对象引用是为null。值可以被分配在声明或构造函数内。

  • 实例变量可以通过调用内部类的变量名直接访问。然而，在静态方法和不同的类（实例变量时给出可访问）使用完全合格的名称应该叫ObjectReference.VariableName.

例子: 

import java.io.*;

class Employee{
   // this instance variable is visible for any child class.
   public String name;
   
   // salary  variable is visible in Employee class only.
   private double salary;
   
   // power by www.yiibai.com
   // The name variable is assigned in the constructor. 
   public Employee (String empName){
      name = empName;
   }

   // The salary variable is assigned a value.
   public void setSalary(double empSal){
      salary = empSal;
   }
   
   // This method prints the employee details.
   public void printEmp(){
      System.out.println("name  : " + name );
      System.out.println("salary :" + salary);
   }

   public static void main(String args[]){
      Employee empOne = new Employee("Ransika");
      empOne.setSalary(1000);
      empOne.printEmp();
   }
}

这将产生以下结果: 

name  : Ransika
salary :1000.0

类/静态变量 :

  • 静态变量的类变量是同一个类中的static关键字声明，但外面的方法，构造函数或块为类实例.

  • 只有每个类的类变量的一个副本，无论从它创建多少个对象。

  • 被声明为常数以外，很少使用静态变量。常量是作为公共/私有的，最终的和静态变量声明。常数变量不会改变其初始值.

  • 静态变量存储在静态内存。这是罕见的，使用静态变量比声明最后，作为公共或私有常数.

  • 静态变量是在程序启动时创建和销毁的程序停止时.

  • 可见性是类似于实例变量。然而，大多数静态变量声明为public，因为他们必须为类的用户提供.

  • 默认值是相同的实例变量。对于数字的默认值是0，布尔值，它是false，对象引用是为null。值可以被分配在声明或构造函数内。此外，可以被分配在特殊的静态初始化块值.

  • 静态变量可以访问通过调用类的名称.ClassName.VariableName.

  • 当声明类的public static final，那么变量的名称（常量）是大写的变量。如果静态变量是不公开和最后的命名语法是相同的实例和局部变量.

实例: 

import java.io.*;

class Employee{
   // salary  variable is a private static variable
   private static double salary;

   // DEPARTMENT is a constant
   public static final String DEPARTMENT = "Development";

   public static void main(String args[]){
      salary = 1000;
      System.out.println(DEPARTMENT+"average salary:"+salary);
   }
}

这将产生以下结果: 

Development average salary:1000

注: 如果变量是从一个外部类访问常数应作为Employee.DEPARTMENT访问
