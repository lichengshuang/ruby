Java对象和类

Java对象和类编程教程,例子教程,参考手册和代码-Java是一种面向对象语言。作为一种语言，具有面向对象特征，Java支持以下的基本概念。

Java是一种面向对象语言。作为一种语言，具有面向对象特征，Java支持以下的基本概念:

  • 多态性(Polymorphism)
  • 继承权(Inheritance)
  • 封装(Encapsulation)
  • 抽象化(Abstraction)
  • 类(Classes)
  • 对象(Objects)
  • 实例(Instance)
  • 方法(Method)
  • 消息解析(Message Parsing)

在本章中，我们会考虑类和对象的概念.

  • 对象 - 对象具有状态和行为。例如：狗有颜色，名称，品种以及行为摇摆，吠叫，吃。对象是类的一个实例.

  • 类 - 一个类可以被定义为描述行为/状态模板/蓝印花布，其类型支持的对象.

Java对象:

让我们更深入看看什么是对象。如果我们考虑现实世界中，我们可以发现我们周围的许多对象，汽车，狗，人类等所有这些对象的状态和行为.

如果我们考虑了一只狗，那么它的状态。名称，品种，颜色和行为。吠叫，摇摆，跑步

如果你比较与现实世界对象的软件对象，他们有非常相似的特点.

软件对象也有状态和行为。软件对象的状态被存储在字段和行为是通过方法所示。
所以操作一个对象的内部状态，并在软件开发方法的对象到对象的通信是通过方法做.

Java类:

一类是从个别对象的创建蓝图(blue print).

下面给出一类的例子: 

public class Dog{
   String breed;
   int age;
   String color;

   void barking(){
   }
   
   void hungry(){
   }
   
   void sleeping(){
   }
}

一个类可以包含任何以下变量类型.

  • 局部变量. 方法，构造或块内定义的变量称为局部变量。该方法已经完成时声明和初始化变量将在方法和变量将被销毁时.

  • 实例变量 . 实例变量是变量在类以外的任何方法。这些变量被实例化的类被加载时。实例变量可以从特定类别的任何方法，构造或块内.

  • 类变量. 类变量以外的任何方法，在一个类中声明的变量，static关键字.

一个类可以有任意数量的方法来访问各类方法的价值。在上面的例子，如：barking(), hungry() 和 sleeping() 都是变量.

下面提到的一些需要讨论的重要主题Java语言类.

构造方法:

类讨论的最重要的子课题之一是构造。每个类有一个构造方法。如果我们不明确地写了一个类的构造Java编译器建立一个类的默认构造函数.

每创建一个新的对象至少有一个构造时将被调用。构造函数的主要规则是，他们应该具有与类相同的名称。一个类可以有多个构造函数.

例子下面给出一个构造: 

class Puppy{
   public puppy(){
   }

   public puppy(String name){
      // This constructor has one parameter, name.
   }
}

Java也支持单例类的地方，你就可以创建一个类只有一个实例.

创建对象:

正如前面提到的一类为对象提供的蓝图。所以基本上创建一个对象从一个类。在Java中的新的关键的词是用来创建新的对象。
从类创建一个对象时，有三个步骤:

  • 声明.  一个对象类型的变量声明变量名.

  • 实例 . 用于创建对象的'new'的关键字.

  • 初始化. 随后通过调用o构造的'new'的关键字。这个调用初始化新对象.

创建对象的实例下面给出: 

class Puppy{
   public Puppy(String name){
      // This constructor has one parameter, name.
      System.out.println("Passed Name is :" + name ); 
   }
   public static void main(String []args){
      // Following statement would create an object myPuppy - by www.yiibai.com
      Puppy myPuppy = new Puppy( "tommy" );
   }
}

如果我们编译并运行上述程序，那么这将产生以下结果: 

Passed Name is :tommy

访问实例变量和方法:

通过创建对象的实例变量和方法的访问。要访问一个实例变量的完全限定路径应该如下: 

/* First create an object */
ObjectReference = new Constructor();

/* Now call a variable as follows */
ObjectReference.variableName;

/* Now you can call a class method as follows */
ObjectReference.MethodName();

例子:

这个例子说明了如何访问一个类的实例变量和方法: 

class Puppy{
   
   int puppyAge;

   public Puppy(String name){
      // This constructor has one parameter, name. by www.yiibai.com
      System.out.println("Passed Name is :" + name ); 
   }
   public setAge( int age ){
       puppyAge = age;
   }

   public getAge( ){
       System.out.println("Puppy's age is :" + puppyAge ); 
       return puppyAge;
   }
   public static void main(String []args){
      /* Object creation */
      Puppy myPuppy = new Puppy( "tommy" );

      /* Call class method to set puppy's age */
      myPuppy.setAge( 2 );

      /* Call another class method to get puppy's age */
      myPuppy.getAge( );

      /* You can access instance variable as follows as well */
      System.out.println("Variable Value :" + myPuppy.puppyAge ); 
   }
}

如果我们编译并运行上述程序，那么这将产生以下结果: 

Passed Name is :tommy
Puppy's age is :2
Variable Value :2

源文件声明规则:

作为本节的最后一部分，现在让我们看到源文件声明规则。这些规则是必不可少的，当在源文件中声明类，import语句和包声明.

  • 每个源文件只能有一个public类.

  • 一个源文件可以有多个非公共类.

  • 公共类的名称应该是源文件的名称，以及应在末尾追加.java。例如：类名。公共类Employee{}，然后源文件应该Employee.java.

  • 如果类定义一个包内，然后包语句应该在源文件中的第一条语句.

  • 如果import语句存在，那么他们之间的package语句和类声明必须写。如果没有包声明，import语句应该是在源文件中的第一行.

  • 导入(Import)和包声明将意味着目前在源文件中的所有类。这是不可能的声明在源文件中的不同阶层不同的导入和/或包声明.

类具有几个访问级别，有不同类型的类，抽象类，final类等我将所有这些访问修饰符一章中解释.

除了上面提到的类型的类，Java还具有一些特殊的类称为内部类和匿名类.

Java 包:

简单的，它是一个分类的类和接口的方式。在开发Java应用程序中，数百的类和接口将被写入，因此，归类这些类是必须的，以及使轻松得多.

Import 语句:

如果给出一个完全合格的名称，其中包括包和类的名称，然后在Java编译器可以很容易地找到源代码或类。 import语句是给编译器的正确位置的方法，找到特定的类.

例如行会要求编译器加载Java安装目录中的所有类java/io : 

import java.io.*;

一个简单的例子学习:

对于我们的案例研究，我们将创建两个类。他们是Employee 和EmployeeTest.

首先打开记事本，并添加下面的代码。记住，这是Employee类和类是一个公共类。现在保存这个源文件的名称Employee.java.

Employee类的有4类变量name, age, designation 和salary。这个类有一个明确定义的构造函数需要一个参数. 

import java.io.*;
public class Employee{
   String name;
   int age;
   String designation;
   double salary;
	
   // This is the constructor of the class Employee
   public Employee(String name){
      this.name = name;
   }
   // Assign the age of the Employee  to the variable age.
   public void empAge(int empAge){
      age =  empAge;
   }
   /* Assign the designation to the variable designation.*/
   public void empDesignation(String empDesig){
      designation = empDesig;
   }
   /* Assign the salary to the variable	salary.*/
   public void empSalary(double empSalary){
      salary = empSalary;
   }
   /* Print the Employee details - by www.yiibai.com */
   public void printEmployee(){
      System.out.println("Name:"+ name );
      System.out.println("Age:" + age );
      System.out.println("Designation:" + designation );
      System.out.println("Salary:" + salary);
   }
}

如前面提到在本教程中处理，从主体(main)方法。因此，在我们运行这个Employee类的顺序应该是main的方法，应该创建对象。我们将创建一个单独类为这些任务.

下面给出的是EmployeeTest类而创建的Employee类的两个实例，并调用每个对象的方法，为每个变量指定值。
EmployeeTest.java文件保存下面的代码: 

import java.io.*;
public class EmployeeTest{
   public static void main(String args[]){
      /* Create two objects using constructor */
      Employee empOne = new Employee("James Smith");
      Employee empTwo = new Employee("Mary Anne");

      // Invoking methods for each object created. by www.yiibai.com
      empOne.empAge(26);
      empOne.empDesignation("Senior Software Engineer");
      empOne.empSalary(1000);
      empOne.printEmployee();

      empTwo.empAge(21);
      empTwo.empDesignation("Software Engineer");
      empTwo.empSalary(500);
      empTwo.printEmployee();
   }
}
