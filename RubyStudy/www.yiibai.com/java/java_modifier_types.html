访问控制修饰符:

Java提供了一个访问修饰符设置类，变量，方法和构造方法的访问级别。四个访问级别:

  1. 程序包可见。默认，任何修改是必要的.

  2. 类可见（私有）.

  3. 所有可见（公共）.

  4. 可包和所有的子类见（保护）.

非访问修饰符:

Java提供了一些非访问修饰符来实现许多其他功能.

  • 创建类的方法和变量的static修饰符

  • final修饰符的类，方法和变量的实现，最终确定。

  • 创建抽象类和方法使用abstract修饰符。

  • synchronized和volatile修饰词，这是线程使用。.
