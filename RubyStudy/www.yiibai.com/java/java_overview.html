Java概述

Java编程语言最初是由Sun Microsystems，由James Gosling和发起的：Sun Microsystems的Java平台（Java 1.0版的J2SE）的核心组件发布于1995年。

Java编程语言最初是由Sun Microsystems，由James Gosling和发起的：Sun Microsystems的Java平台（Java 1.0版的J2SE）的核心组件发布于1995年。.

截至12月08日最新版本的Java标准版（J2SE）的。 Java和其广泛的蔓延流行与进步，多种配置，内置套件各类平台。例如：J2EE企业应用，J2ME移动应用.

Sun Microsystems已更名为新的Java SE，Java EE和Java ME的分别J2的版本。 Java是保证一次编写，随处运行。

Java是:

  • 面向对象 :  在java中一切都是对象。 Java可以很容易地扩展，因为它是基于对象模型.

  • 独立于平台: 不像许多其他的编程语言，包括C和C++编译Java时，它被编译成平台特定的机器，而是把平台无关的字节代码，。这个字节码分布在网络平台正在运行的虚拟机（JVM）的解释.

  • 简单 :  Java的设计是简单易学。如果你理解Java的OOP的基本概念很容易掌握.

  • 安全 :  与Java的安全功能，它使开发无病毒，无篡改系统。认证技术是基于公共密钥加密.

  • 体系结构中性 :  Java编译器生成体系结构中立的目标文件格式，这使得编译后的代码是对多处理器的可执行文件，存在的Java运行时系统.

  • 可移植 :  使得Java的便携式体系结构中立，没有规范的实施依赖各个方面。 ANSI C编写，编译器和Java用干净的可携性的边界，这是一个POSIX子集.

  • 健壮性 :  Java使得致力，以消除容易出错的情况，主要强调编译时错误检查和运行检查.

  • 多线程 :  随着Java的多线程功能，它可以编写程序，可以同时做很多工作。此设计功能允许开发建设顺利运行的交互式应用程序.

  • 解释 :  Java字节码翻译上的本地机器指令飞不存储任何地方。因为链接是一个渐进，重量轻过程，发展过程中更加快速和分析.

  • 高性能:  使用一次编译Java使高性能.

  • 分布式 :  Java被设计为互联网的分布式环境.

  • 动态 : Java被认为是更具活力比C或C ++，因为它的设计，以适应不断变化的环境。 Java程序可以进行广泛的数量，可以用来验证和解决在运行时访问对象的运行时信息.

Java的历史:

James Gosling(詹姆斯·高斯林)发起于1991年6月，在他的许多机顶盒项目使用Java语言项目。语言，最初称为Gosling的办公室外站后，一棵橡树，橡树，也去的名称为绿色和结束后改名为Java随机单词列表，.

Sun发布Java 1.0的首次公开实施，在1995年。它承诺一次编写，随处运行（WORA）的，流行的平台上提供无成本的运行时.

2006年11月13日，Sun发布了GNU通用公共许可证（GPL）的条款下自由和开放源码软件的Java.

2007年5月8日Sun完成这个过程，使所有Java的核心代码，免费和开源，除了小部分的代码，而的Sun没有拥有版权.
